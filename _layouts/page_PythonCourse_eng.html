Course Info

In this course, students will learn the programming language Python for their careers in healthcare industries. The course covers an introduction to the basic language syntax, the concept of object-oriented programming, and the usage of powerful packages. This course aims to develop students' capability of programming in terms of understanding given pieces of code and implementing their programs for entry-level tasks and data analysis. This course includes lectures and coding practice, a midterm exam, and a final project with a presentation. Because there are plentiful resources for self-studying Python through books and video tutorials, students are encouraged to learn on their own and seek help from me and the TA.

Requirement

High-school-level mathematics

Grade

10% Exercise, 30% Midterm, 45% Final project, 15% Attendance

Textbook

Not assigned because there are many books explaining the syntax of the Python language, even the Python Foundation's online tutorial helps. Python 3 has been released for years, and the syntax changes little in recent years. Most books published after 2017 still work in 2024. The only significant added syntax after Python 3.8 is the match-case statements, and all the remaining ones are unlikely to be used in our course.

School library. You can find 300+ English Python books there.
